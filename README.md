# Hoe open je een oefening.

Om een oefening te openen, gebruik je niet de map **oo-dev** als hoofdmap, maar de map van de specifieke oefening.

## Een voorbeeld.

Om de practice-oefening van 2. classes, part 1, te openen, ga je naar de volgende locatie:
**oo-dev/2-Classes-part-1/practice/example-bicycle**
